================Wizard's Tower===============
Requires Unity 5.5.0 or higher.



This pack includes model of fantasy Wizard's Tower (Interior + Exterior): 
- Cloud Tower (DemoScene 1);
- Ground Tower (DemoScene 2);
- Some Props (Globe, Book, Magic Wand, Sand Clock, Wallframe, Picture Frame);
- Clouds (Particle based); 
- PBR Textures (2K).

Tris count:
- Tower (interior + exterior) - 18007 tris;
- Cloud platform - 6058 tris.
